# Numerical Approximation of Electromagnetic Field Integrals Using MATLAB

## Project Overview

This repository contains the MATLAB implementation, analysis, and supporting documentation for a computational engineering research project investigating the numerical approximation of electromagnetic field integrals.

The project was conducted as part of the Numerical Methods II course in Electrical and Electronic Engineering and explores how numerical integration techniques can be applied to electromagnetic field problems where analytical solutions may be difficult, expensive, or impractical to obtain.

The study focuses on modeling the magnetic field generated by a current-carrying conductor and evaluating the performance of different numerical integration methods through computational experimentation.

---

## Research Objective

The primary objective of this project was to investigate the effectiveness of numerical integration methods for approximating electromagnetic field integrals and to compare their computational accuracy, convergence behavior, and numerical stability.

Specifically, the project sought to:

* Model magnetic field behavior around a current-carrying conductor.
* Approximate electromagnetic field integrals using numerical methods.
* Compare numerical results against analytical solutions.
* Evaluate convergence characteristics and computational error.
* Assess the suitability of different numerical integration approaches for engineering applications.

---

## Numerical Methods Implemented

### Composite Trapezoidal Rule

The Composite Trapezoidal Rule was implemented to numerically approximate the electromagnetic field integral by partitioning the integration domain into smaller intervals and applying piecewise linear approximations.

### Composite Simpson's Rule

The Composite Simpson's Rule was implemented to improve approximation accuracy by using quadratic interpolation across integration intervals.

---

## Methodology

The project workflow consisted of the following stages:

1. Mathematical formulation of the electromagnetic field integral.
2. Development of MATLAB algorithms for numerical approximation.
3. Implementation of Composite Trapezoidal Rule.
4. Implementation of Composite Simpson's Rule.
5. Analytical evaluation of the exact solution.
6. Comparison of numerical and analytical results.
7. Error computation and convergence analysis.
8. Visualization of numerical behavior using MATLAB-generated plots.

---

## Key Findings

The computational experiments demonstrated that:

* Both numerical methods successfully approximated the electromagnetic field integral.
* Approximation accuracy improved as the number of integration intervals increased.
* Simpson's Rule consistently achieved higher accuracy than the Trapezoidal Rule for the smooth electromagnetic field functions considered.
* Simpson's Rule exhibited faster convergence toward the analytical solution.
* Numerical error decreased predictably with mesh refinement.

These results highlight the importance of selecting appropriate numerical methods when solving engineering problems involving electromagnetic analysis and simulation.
